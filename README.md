# Finanzas RNM

FinanzApp Pro — Panel de administración para registrar y analizar finanzas personales.

## Sitio
Cuando GitHub Pages esté activado, estará disponible en:
https://RdrgNoc.github.io/finanzas-rnm/

## Cómo usar
- Abre el sitio y registra ingresos/egresos.
- Administra categorías y métodos de pago desde Configuración.
- Exporta/Importa respaldos (JSON/CSV) desde Ajustes.

> Nota: La app guarda datos en el navegador (localStorage) por usuario/dispositivo.
